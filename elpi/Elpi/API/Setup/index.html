val init : ?⁠flags:flags -> builtins:builtins list -> ?⁠file_resolver:(?⁠cwd:string -> file:string -> unit -> string) -> unit -> elpi

Initialize ELPI.

init must be called before invoking the parser.

parameter flags

for the compiler, see Compile.flags

parameter builtins

the set of built-in predicates, eg Elpi.Builtin.std_builtins

parameter file_resolver

maps a file name to an absolute path, if not specified the options like -I or the env variable TJPATH serve as resolver. The resolver returns the abslute file name (possibly adjusting the file extension). By default it fails. See also Parse.std_resolver.

returns

a handle elpi to an elpi instance equipped with the given builtins and where accumulate resolves files with the given file_resolver.
